## Development

### Building dependencies
```bash
yarn install
```

### Starting Dev Server
```bash
yarn start
```

### Building
```bash
yarn build
```

### Running Tests
```bash
yarn test
```

### Deploying to `gh-pages`
```bash
yarn deploy
```
