A front end to utilise AnyAudio API.

[![Build Status](https://travis-ci.org/anyaudio/anyaudio-web-app.svg?branch=master)](https://travis-ci.org/anyaudio/anyaudio-web-app)


## Development

### Building dependencies
```bash
yarn install
```

### Starting Dev Server
```bash
yarn start
```

### Building
```bash
yarn build
```

### Running Tests
```bash
yarn test
```

### Deploying to `gh-pages`
```bash
yarn deploy
```
